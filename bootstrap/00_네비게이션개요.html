navbar 기본 클래스 설명

        .navbar 네비게이션 바를 정의하는 기본 설정
            - 네비게이션 바의 기본 설정 스타일을 적용
          
        .nabar-expand{-sm -md -lg -xl -xxl}
            - 네비게이션 바가 특정 화면  크기 이상에서 확장되도록 함
            - 화면 크기가 설정된 값보다 작을 때, 햄버거 메뉴 아이콘으로 축소
            - sm : 작은 화면 크기 576px(아이패드 미니 정도) 이하는 햄버거 메뉴 아이콘
            - md : 중간 화면 크기 768px (갤탭,아이패드) 이하는 햄버거 메뉴
            - lg : 큰 화면 크기 992px 햄버거 메뉴 아이콘 보임 작은 모니터
            - xl : 매우 큰 화면 크기 1200px 이상만 제대로 표시
            - xxl : 1400px 이상
        .navbar-light :
            밝은 배경색 네비게이션 바
            *폰트 어두움

        .navbar-dark :
            *폰트 밝음. 무조건 bg-dark 혹은 primary 같이 배경색을 변경해줘야 한다.
            .bg-light : 네비게이션 바 배경색 밝게
            .bg-dark : 네비게이션 바 배경색 어둡게
            .bg-primary : 네비게이션 바 배경색 기본색상

        .navbar-brand : 네비게이션 바의 브랜드/로고 표시 역할
        .navbar-toggler : 작은 화면에서 네비게이션 바를 열고 닫는 버튼
        버튼의 기능은 아무것도 없는 상태이며 열고 닫는 기능만 존재하고, 디자인적요소가 없다.
        .navbar-toggler-icon : 눈에 보이는 디자인 icon을 추가. (default 햄버거 모양)
        .collapse : 반응형으로 동작하는 내용을 포장
        .navbar-collapse : 네비게이션 바의 내용을 축소/확장 가능하도록 선언
        .navbar-nav : 네비게이션 바 내부 목록을 감싸는 부분
        .nav-item : 네비게이션 바 각 항목을 표현
        .nav-link : 네비게이션 바의 각 항목의 링크를 디자인 한 것
        data-bs-toggle = "collapse"
            부트스트랩에서 js 사용해서 태그를 토글로 만드는 역할
        data-bs-target = "#아이디명"
            토글 대상의 ID를 나타냄
