공간 padding 여백 margin

        Padding - 차지하는 범위 내 여유 공간 설정
            pt : 태그 위쪽 패딩(padding-top)을 설정
            pr : padding-rgiht
            pb : padding-bottom
            pl : padding-left
            
            px : 좌우 패딩
            py : 상하 패딩
        Margin - 다른 태그/값과 거리를 두는 것
        mt : 태그 위쪽 패딩(margin-top)을 설정
        mr : margin-rgiht
        mb : margin-bottom
        ml : margin-left
        
        mx : 좌우 margin
        my : 상하 margin
        m- 태그에 모든 방향 여백을 설정

숫자값

        0,1,2,3,4,5,auto 사용 가능. 홈페이지 주로 mt-5 한다
        0: 여백/패딩 없음
        1: 25rem(=약 4px)
        2: 5rem(=약 8px)
        3: 1rem(=약 16px)
        4: 1.5rem(=약 24px)
        5. 3rem(=약 48px)

        -------이게 뭐야?
        (==25rem) 0.25rem 계산 > 0.25 *16px = 4px
        (==5rem) 0.5rem 계산 0.5 * 16px = 8px
        1 rem 계산 1 * 16px = 16px
        1.5rem 계산 1.5 * 16px = 24px
        3 rem 3 * 16px = 48px
