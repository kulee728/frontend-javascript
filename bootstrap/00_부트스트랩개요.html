            col-(xs,md,lg,xl)-1 : 사이즈 지정 1칸

card class 구성

        card: 카드 스타일 적용
        card-body : 카드 본문 틀 안에 img 나 title이나 text등을 넣어줌
        card-title : 카드 본문에서 제목을 나타냄
        card-text : 카드 본문 텍스트 내용
        card-img-(top 또는 bottom 또는...위치) : card 내 삽입된 이미지의 위치 적용
        >> img 가 카드 바깥으로 나가는것을 방지, 카드 틀 내 이미지가 있을 수 있도록 도와준다.
    
        img 설정하기 (클래스들)
        img-fluid : col이나 가로길이에 맞도록 이미지 크기를 조절(fluid : 액체)
            col-4 img-fluid 는 4칸 기준으로 이미지 크기가 딱 맞도록
            width :100%, height : auto 로 된다
        img-thumbnail : 이미지에 테두리와 패딩을 추가, 썸네일 느낌 나도록 한다.
            rounded : 이미지 모서리 둥글게 만듬
            rounded-circle : 이미지를 원형으로 만든다
            rounded-pill : 이미지 모서리를 완전히 둥글게
    
        d-flex : 태그를 flex 컨테이너로 만들어줌
            태그 아래에 생기는 아이템이나 태그는 flex에서 지정한 규칙에 따라
            정렬한다.
        justify-content-end : flex 컨테이너 내에서
            아이템이나 태그를 오른쪽 끝으로 정렬시킨다
        justify-content-start : flex 컨테이너 내에서
            아이템이나 태그를 왼쪽 끝으로 정렬시킨다
        justify-content-center : flex 컨테이너 내에서
            아이템이나 태그를 가운데 정렬시킨다
        
        table-striped : 테이블 행이 번갈아 다른 배경색 적용시키기.
            시각적으로 행간 구별이 편해진다
