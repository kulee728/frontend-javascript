변수 (Variable)

        - 메모리의 값을 계속 바꿔가며 저장 할 수 있는 공간
        - JS의 변수명은 영문자,숫자,'_','$'
        단, $ 표시는 jQuery 표기 문법으로 사용할 예정이며 따라서
        js 작성시에는 사용을 지양한다
        - 카멜 표기법 사용

JavaScript 변수 종류와 특징 

        JS의 변수 자료형
        var : 변수 중복X 
        let : 지역변수. 중복 X
        const : 읽기 전용 상수. 변수명 중복 x
        ...우리는 const 만 주로 쓸 것이다
