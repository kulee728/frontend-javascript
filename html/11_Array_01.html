JavaScript의 배열

    - 자료형의 제한이 없다
    - 배열의 크기가 가변적이다 (제한이 없다)
    예시 :
    const arr = new Array(); //0칸짜리
    const arr = new Array(3); //3칸짜리 배열
    const arr = [] //비어있는 배열 생성
    const arr = [값1, 값2, 값3] 
    이차원 배열은

    js array 관련 기능
    arr.length 
    arr.fill
    arr.find , findIndex , sort, ...
