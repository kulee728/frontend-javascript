정규표현식

        - 정규식 (정규표현식)
        - 특정한 규칙을 가진 문자열 집합을 표현하는 형식언어
        - 입력된 문자열에 대한 조건 검색, 일치여부, 필터링을 간단하게 처리

        정규표현식의 생성
        1) const regExp = /^정규표현식$/;
        (/^은 시작, $/은 종료를 의미한다)

        2) const regExp = new RegExp("정규표현식");
        
        
        문자열 패턴 확인
        1) regExp.test(str)
            > str이 정규표현식과 일치하는 패턴인경우 true, 없다면 false
        2) regExp.exec(str)
            > str에 정규표현식과 일치하는 패턴인 경우 
            처음 매칭되는 문자열 반환, 없으면 null반환

정규표현식 문자 매칭 방법

        [] : 대괄호 안의 어떤 문자와 일치
        /^[a,b,c,d]$/ : abcd 중 하나라도 일치하는 문자 있는지
        [0-9] : 0부터 9까지
        [ㄱ-힇] : ㄱ부터 힇까지 모든 한글
        [가-힇] : 가 부터 힇까지 모든 글자
        [a-z] : 모든 영어 소문자
        [A-Z] : 모든 영어 대문자
        --------------------------------------------------------
        [^]대괄호 안의 문자와 일치하지 않음 (!와 같은 역할)
        [^abcd]가 하나라도 있으면 false
        [A-Z]{5} : A-Z사이 5자리
        [a-z]{2,5} : a-z사이 2~5자리
        [A-Z]{10,} : 10자리 이상
        --------------------------------------------------------
        () : 그룹을 형성하고 일치하는 부분 문자열을 기억
            ex. (19|20) 19나 20과 일차하는
    
        \d : 0~9 중 아무 숫자
        \s : 띄어쓰기, 엔터, 탭 등 공백 문자
