attr

        attr attribute 속성
        jQuery 에서 HTML 태그 속성을 가져오거나 설정할 때 사용
        예제
        
        
        
        src alt style class id 전부 attribute 속성값이다.

        사용 1. set
        $("그림클래스").attr('alt',"설명바꾸기");

        +)여러개 한꺼번에 set
        $("그림클래스").attr(
            'alt':"설명바꾸기",
            'src':"바뀐경로",
            ...
            
        );

        사용 2. get
        const getAlt =  $("그림클래스").attr('alt');
         
        ///
        

        jQuery가 없었다면
        getAttribute
        setAttribute를 사용했을 것이다.
