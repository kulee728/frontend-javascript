jQuery

        자바스크립트를 간편하게 사용할 수 있도록 하는
        오픈소스 기반 자바스크립트 라이브러리

        사용 방법 1- 홈페이지 다운로드 src="jquery-3.7.1.min.js"

        사용 방법 2- CDN 주소 참조 활용 (Contents Delivery Network)
        script 에 type="text/javascript" src="https://code.jquery.com/jquery-3.7.1.min.js"
        를 속성으로 넣어준다.
        


        [기본 문법]

            })
        })

        [선택자 문법]
        
        1. css 선택
        $("태그명, #아이디명, .클래스명").css("style속성",설정할값);

        2.toggle
        $("태그명, #아이디명, .클래스명").toggle("속도조절"); //속도는 자유롭게, 생략도가능

        3. slideToggle 선택 사용
            주로 이미지를 대상으로 사용
        
        4. hide() show() 선택 사용
            숨기기 보이기

        5. append() 태그나 값을 추가할 때 사용

        6. val()
            input textarea select 같은 폼안에 들어가는 태그의 값을 가져오거나
            설정하는데에 사용
            값을 가져올 때 $(태그명, #아이디값, .클래스명).val();
                ->$("input").val()
                ->$("#idName").val()
                ->$(".className").val()
            값을 넣을 때는 $(태그명, #아이디값, .클래스명).val("새로운 값");
            
        7. this 
         현재 실행중인 함수 문맥

        8. on
        이벤트 기능 : 하나 이상의 이벤트를 처리
        $(태그명, #아이디값, .클래스명).on(event,function(){
            
        })
        - event 유형 : click input mouseover mouseenter
        >> on은 이벤트가 많아 on("input",function(){

        }) 처럼 이벤트명을 명시해주어야 하지만
        다른 속성들은 이벤트명이 보통 명시가 없다.


        9. keyup()
        - 사용자가 입력 창(input textarea,...)에 키를 입력할 때 마다
        발생하는 이벤트릴 감지한다. 키가 눌릴 때 마다 호출된다

        10. .each()

        $("li").each(function(){
            //li 태그에 대해 반복적으로 내용을 수행한다.
            //this로 현재 태그에 대해 접근도 할 수 있다.
        })

        11. 긴묶음.indexOf(str)
        -str을 포함하고 있는지, 포함한다면 그것의 첫번째 인덱스를 반환
        -찾지 못하면 -1을 반환
        - 배열에도 사용 가능하다. arr.indexOf(searchingValue) 처럼
